Assessment Guidance

Perform a systematic head‑to‑toe physical assessment. For each system: state normal findings, identify deviations, and document supportive data. Mark PASS if assessed accurately & documented; FAIL if omitted, incomplete, unsafe, or incorrect interpretation. Critical elements reflect safety or core neurological/respiratory status.
States nutrition & habitus (well-nourished / proportionate / no distress)
Describes comfort / no acute distress / appropriate posture
